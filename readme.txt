== Description ==

The plugin is for simplifying installation of [Jumpout](http://makedreamprofits.com/jo/) popups on your website.


** The plugin is only for the customers of JumpOut product**

== Installation ==

After you will install the plugin, simply find it in the left menu and follow the instructions it provides. If you have any problems, [contact our support](http://makedreamprofits.com/support/) .


== Frequently Asked Questions ==

= All people can use it, isn't it? =

No, only who bought JumpOut popups. But, we have a [trial](http://makedreamprofits.com/jo/video/trial/)
